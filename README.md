# Project Description
This is a Task Management Web Application built using MERN stack (MongoDB, Express.js, React.js, Node.js). 

# Setup Instructions
To run this project locally:

1. Clone the repository.
2. *Backend Setup:*
   - cd backend
   - npm install
   - npm start
3. *Frontend Setup:*
   - cd frontend
   - npm install
   - npm run dev

## Features
- User Registration & Login
- CRUD Operations (Add, Edit, Delete, View tasks)
- Mark tasks as completed or pending
