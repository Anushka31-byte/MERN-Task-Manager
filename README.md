# MERN Task Management Application

*Student Name: Anushka Vikas Deshmukh*

## Project Description
This MERN Task Management App enables users to securely Register and Login to their accounts. Users can perform CRUD operations: Create, Read, Update, and Delete tasks, helping them stay organized with a clear overview of their daily progress.
